# 项目管理功能 UI 设计稿

## 背景

当前创建群聊时需要手动输入项目目录路径，存在以下问题：

1. 每次创建群聊都要重复输入相同的项目路径
2. 路径容易输错，且没有统一管理
3. 无法复用项目配置

**解决方案**：引入"项目"概念，先创建项目配置，再基于项目创建群聊。

## 7. 数据模型

### Project 模型

```typescript
interface Project {
  id: string; // 唯一标识
  name: string; // 项目名称（可读的，唯一，不可更改）
  directory: string; // 项目目录（绝对路径）
  documents: string[]; // 项目文档路径数组
  description?: string; // 项目描述
  rules: ProjectRule[]; // 项目规则列表
  createdAt: number; // 创建时间戳
  updatedAt: number; // 更新时间戳
}
```

### ProjectRule 模型

```typescript
interface ProjectRule {
  id: string; // 规则唯一标识
  projectId: string; // 所属项目ID
  title: string; // 规则标题
  content: string; // 规则内容（Markdown 格式）
  createdAt: number; // 创建时间戳
  updatedAt: number; // 更新时间戳
}
```

```typescript
interface GroupSessionMeta {
```

## 8. 交互细节

### 8.1 路径选择器

- **输入框右侧图标**：点击打开文件浏览器对话框
- **路径自动补全**：输入时显示匹配的历史路径或文件系统路径
- **路径验证**：实时验证目录/文件是否存在
- **拖拽支持**：支持从文件管理器拖拽文件夹到目录输入框

### 8.2 项目选择体验

- **搜索过滤**：支持按项目名称或路径搜索
- **最近使用**：优先显示最近使用的项目
- **键盘导航**：支持上下方向键选择，Enter确认
- **空状态提示**：没有项目时引导用户创建

### 8.3 创建流程

```
创建群聊入口
    │
    ▼
Step 1: 选择项目
    │
    ├─→ 选择已有项目
    │
    └─→ 选择"不关联项目"（独立群聊）
    │
    ▼
Step 2: 完善群聊信息
    │
    ▼
创建成功 → 进入群聊
```

### 8.4 错误处理

- **目录不存在**：红色边框 + 错误图标 + 文字提示
- **文件不存在**：在文档列表中标注 ⚠️
- **名称重复**：实时验证，显示"项目名称已存在"
- **网络错误**：Toast 提示 + 重试按钮

## 12. 管理按钮功能设计

### 12.1 设计背景

项目不仅是一个配置容器，更是群聊协作的核心上下文来源。当多个 Agent 在群聊中协作时，项目提供：

- **工作空间**：CLI Agent 的工作目录
- **知识库**：注入到上下文的文档
- **协作基准**：共享的项目配置和规范

### 12.8 数据模型扩展

```typescript
// 扩展 Project 类型
interface Project {
  id: string;
  name: string;
  directory: string;
  documents: string[];
  description?: string;
  createdAt: number;
  updatedAt: number;

  // 新增字段
  /** 项目级提示词，注入到所有关联群聊的 Agent 上下文 */
  projectPrompt?: string;
  /** Agent 职责配置 */
  agentConfigs?: ProjectAgentConfig[];
  /** CLI Agent 工具策略 */
  toolPolicy?: ProjectToolPolicy;
}

interface ProjectAgentConfig {
  agentId: string;
  /** 默认角色 */
  defaultRole: "assistant" | "member" | "cli";
  /** 职责提示词 */
  rolePrompt?: string;
  /** 自定义工作目录（仅 CLI Agent） */
  customDirectory?: string;
}

interface ProjectToolPolicy {
  /** 允许的文件操作 */
  fileOps?: ("read" | "write" | "mkdir" | "unlink")[];
  /** 允许的终端命令 */
  shellOps?: ("exec" | "chmod")[];
  /** 允许的网络操作 */
  httpOps?: ("fetch" | "ws")[];
  /** 是否允许危险操作 */
  allowDangerous?: boolean;
}

// 扩展 GroupSessionEntry 类型
interface GroupSessionEntry {
  // ... 现有字段 ...

  /** 关联的项目ID */
  projectId?: string;
  /** 是否继承项目配置更新 */
  inheritProjectUpdates?: boolean;
}
```
